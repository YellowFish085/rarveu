# threejs-sandbox

## Commands

### Install npm modules
```sh
$ npm install
```

### Watch code changes
```sh
$ gulp
```

### Run local server if needed
```
# Run local server on localhost:8080
$ python -m http.server
```

## Requierements
* [node/npm](https://nodejs.org/)
* [gulp](https://github.com/gulpjs/)
* [browserify](http://browserify.org/)
* [python](https://www.python.org/)
